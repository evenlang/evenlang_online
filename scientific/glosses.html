Список глосс

Тут находится список глосс, которые присутствуют в текстах.

Открыть в новой вкладке

1. Существительное

Порядок показателей: основа-AL-NUM-CASE-POSS

1.1 Число

V/C n Непродуктивные (kinship & Сo)
PL -l -r -sal, -tal, -nil/-ŋil, -tyl, -ral

1.2 Падежи

V/C n PL
NOM -∅

4. Глагол

Глагольная словоформа: √-деривации-залог-вид-время/наклонение-лицо&число

Глосса Алломорфы Примеры
AD -w, -u, -m адверсатив-пассив: i-dej ‘войти’ → i-w-dej ‘внести’ → i-wu-w-dej ‘быть внесенным’
MED -b, -p медиопассив: aŋa-daj ‘открыть’ → aŋa-b-daj ‘открыться’
CAUS -w.kan каузатив: ma-daj ‘убить’ → ma-wkan-daj ‘заставить убить’
